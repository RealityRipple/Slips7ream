Section 2.2

KB Article Databases

The Prerequisites and Replacements KB Article Databases contain updated information about the relation between Updates. You can check for new versions of these Databases by clicking "Update Databases" button. New Databases are usually published soon after Patch Tuesday (the second Tuesday of the month).

The Prerequisites Database contains a list of Updates which are required for other Updates to function. Slips7ream also uses this information to determine the order the Updates are integrated during the Slipstream Process.

The Replacements Database lists all the updates that have been replaced by newer updates. Slips7ream will treat these Updates as though they are newer and older versions of the same Update, and prompt you to select between them accordingly if they exist in the Updates or the Integrated Updates lists.
