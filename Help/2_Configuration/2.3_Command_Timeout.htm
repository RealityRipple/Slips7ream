Section 2.3

Command Timeout

This field sets how long SLIPS7REAM will wait for a response from an external process. SLIPS7REAM makes calls to programs such as DISM frequently, and can be told to fail if these calls take too long.

By default, the Command Timeout feature is disabled (with a value of 0), which means that SLIPS7REAM will wait until the program closes on its own before continuing.
