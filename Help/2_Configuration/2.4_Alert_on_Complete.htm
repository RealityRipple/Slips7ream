Section 2.4

Alert on Complete

The Alert on Complete feature lets you play a sound effect when the Slipstream Process completes. This file should be a WAV audio file, although any audio file may be selected. You can test playback of the file by clicking the "" (Play) button on the far right.

By default, SLIPS7REAM uses the Windows "tada.wav" or System Default Alert sound, which can be toggled using the "" (System Default) button.

You can also turn this feature on or off from the Main Window by using the On Completion field.
