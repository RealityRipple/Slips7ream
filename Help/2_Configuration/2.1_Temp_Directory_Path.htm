Section 2.1

Temp Directory Path

The Temp Directory is used for all the hard work done by SLIPS7REAM. When files need to be extracted, copied, or modified for internal use of the program, the Temp Directory is where all the files are stored. SLIPS7REAM automatically clears this directory regularly, so it's important that this directory be empty and unused for any other purpose.

The Temp Directory Path field is automatically set to a folder called Slips7ream within your system TEMP folder. If you wish to set a new folder, you can click "Browse..." to choose a location to use.
