Chapter 3

Slipstream Process

Slipstreaming is a straightforward and tedious process, which is why SLIPS7REAM was created: to do it for you. The exact method the program uses is described below.

  1. SLIPS7REAM begins by merging all of the Image Packages into a single WIM. This is done to remove any Image Packages that have been unchecked and merge any Image Packages that need to be merged.
  2. If the Service Pack checkbox is checked, SLIPS7REAM integrates Service Pack 1 into the Image Packages one by one.
  3. If there are any Features to be enabled or disabled, Integrated Windows Updates to be removed, or Integrated Drivers to be deleted, SLIPS7REAM takes care of them.
  4. Now the real work begins. SLIPS7REAM integrates any Updates or Drivers which have been added to the Updates List.
  5. Once all the other work is done, SLIPS7REAM compresses the WIM and may Split it into pieces if needed.
  6. If the Save to ISO checkbox is checked, SLIPS7REAM's last action is to extract the selected ISO, insert the new WIM, and create an updated ISO in its place.
    • If multiple ISOs are needed, SLIPS7REAM automatically creates them with a simple AutoRun notice that lets the user know the Disc is a Data Disc, not an Install Disc.
    • If any ".clg" files exist in the ISO's "sources" folder, they are automatically removed. If you need to generate catalog files for an Image, please use the Windows System Image Manager from the Automated Installation Kit, available from Microsoft's Download Center free of charge.
