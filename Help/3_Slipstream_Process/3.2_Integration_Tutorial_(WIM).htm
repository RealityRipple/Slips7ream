Section 3.2

Integration Tutorial (WIM)

This tutorial will walk you through integrating Windows Updates (including Internet Explorer) into a Windows Image. If you have trouble with Slips7ream or are a beginner at the Slipstreaming process, you may want to consult this guide.

I suggest you start by setting up a folder for Slipstreaming if you are going to be doing this regularly, so I'll be using the following directory tree for this tutorial:

C:\Users\​Admin\Documents\​Win7_ISO\32-Bit\
You can use any location you want, C:\Users\​Admin\Documents\ is just the path used in the example
IE11\
This folder contains Internet Explorer 11 and its prerequisites (minus ones that are unnecessary when using KB3125574)
Windows6.1-KB2639308-x86.msu
Windows6.1-KB2670838-x86.msu
Windows6.1-KB2729094-v2-x86.msu
Windows6.1-KB2834140-x86.msu
IE11-Windows6.1-x86-en-us.exe
This is the offline installer for Internet Explorer 11 (English). If you want to include Spelling and Hyphenation packages for other languages, you can use their EXEs as well
Updates\
This folder contains the updates that will be integrated. For this example, only three updates will be used
Windows6.1-KB3125574-v4-x86.dt
The Windows 7 Convenience rollup Update
Windows6.1-KB3020369-x86.msu
The prerequisite for KB3125574
IE11-Windows6.1-KB3154070-x86.msu
The latest Cumulative Security Update for Internet Explorer 11 (as of April 2016)
SP1.exe
the 32-Bit Windows 7 Service Pack 1 offline installer
Win7.wim
a copy of INSTALL.WIM from the Windows 7 RTM ISO, which will contain the final version

  1. The first step is to select your source and destination INSTALL.WIM, "Win7.wim".
    Click the "Browse..." button next to the INSTALL.WIM field to select "C:\Users\​Admin\Documents\​Win7_ISO\32-Bit\​Win7.wim". You can also drag-and-drop the file into the field with your mouse.
    Slips7ream will automatically read the INSTALL.WIM file's properties, giving you a list of Image Packages.
  2. You can optionally include or remove Image Packages as needed. All the Updates will be integrated into all of the finalized Image Packages automatically.
    You can also read more package data from the Image Packages if you want to change Features, remove Updates, or just see what's already installed. Having a list of Integrated Updates will also ensure that there aren't version conflicts between Updates.
  3. Next, click the "Service Pack" checkbox to enable the Service Pack field(s). Click "Browse..." next to the Service Pack field and choose "SP1.exe", or drag-and-drop the file into the right field.
    Note that if you have mixed 32-Bit and 64-Bit architectures in your Image Packages list, there will be two fields, one for the x86 Service Pack, one for x64.
  4. After providing the Service Pack, it's time to add some Updates.
    Click on "Add to Updates..." and open the "IE11" folder, then select all of the prerequisite MSU files in this folder and click "Open".
    When all of the selected files have been added to the Updates list, click "Add to Updates..." again to add "IE11-Windows6.1-x86-en-us.exe".
    As before, you can also drag the files (or the "IE11" folder) into the Updates list to add them. You can also re-order the files in the list by dragging one or more selected Updates up or down, or by right-clicking your selection and choosing a menu item.
  5. Practice adding Updates by including the files in the "Updates" folder as well.
  6. If you want to put your selected INSTALL.WIM into an ISO, you can check the Save to ISO checkbox and select a copy of the Windows 7 RTM ISO.
    Note that this will still alter the INSTALL.WIM file's contents, it will simply also place it into an ISO.
  7. Now it's time to begin the Slipstream operation: click "Begin" at the bottom of the window to start the process, then sit back and wait as Slips7ream does its work.
    When the process is complete, you can click the "Open Folder" button to open the folder containing your selected INSTALL.WIM file, "Win7.wim". You can copy this WIM to a USB installer's "/sources/" folder and rename it INSTALL.WIM for quick updating of a flash drive, or store the file for later updating.
