Section 3.3

All-in-One Tutorial

Making an All-in-One installer is easy with SLIPS7REAM! Simply follow the instructions below to create your very own Windows 7 AIO ISO.

  1. Make sure you have your 32-bit and 64-bit images ready, as well as any Service Pack, Update, Language Pack, and Driver files you want to include. See the Integration Tutorial for a good simple file structure.
    If you're integrating Enterprise, do that first - Select your RTM INSTALL.WIM or ISO for Windows 7 x86, then select the Windows 7 x86 Enterprise Edition to merge. Remember, if you use a raw WIM file, you should make a new copy of your original INSTALL.WIM and select that, as SLIPS7REAM will directly integrate Enterprise into the WIM. If you use an ISO, just select a new destination ISO under the "Save to ISO" field.
    Of course, do the same for the 64-bit version after your 32-bit merge is complete.
  2. Select your 32-bit image under "INSTALL.WIM" and your 64-bit image under "Merge WIM", of course choosing a new "Save to ISO" destination for the final ISO, or using a new copy of your 32-bit image if you're working directly with WIM files.
    If you want to include a Service Pack or Updates, now would be a good time. Remember to select both an x86 Service Pack 1.exe and x64 Service Pack 1.exe in the right fields, and both 32 and 64-bit variants of any Windows Updates and/or Drivers you want in all Editions.
  3. Remember to check the Unlock All Editions checkbox, and to select a 32-bit Image if you are saving to an ISO.
    When you're ready, hit the "Begin" button. SLIPS7REAM will automatically merge your 32-bit and 64-bit Image Packages into one WIM, then add whatever other files you have selected. Finally, it will compress the image, and if instructed, create a finalized ISO.
