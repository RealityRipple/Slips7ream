Section 5.13

Service Pack

A Service Pack is a large collection of updates and core Operating System component improvements. They will often replace the need for older updates, and in the case of Service Pack 1 for Windows 7, defunct some Language Packages.

Windows 7 has only one Service Pack, which Slips7ream can integrate into the Windows installer. However, the 64-Bit variant requires a 64-Bit Operating System to be used for the process, because part of the procedure is running the Service Pack EXE to let it extract itself, and 64-Bit programs do not run on 32-Bit Operating Systems.
