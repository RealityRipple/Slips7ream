Section 5.15

Split

A File Split is a collection of two or more files that actually contain contiguous data normally contained in one file. The most common reason to Split a file is because of File System restrictions or Disc space.

Slips7ream can Split a WIM or ISO into sections to fit CDs, DVDs, Blu-Rays, or any custom size. When WIM files are split into pieces, the file type becomes SWM instead, and can be stored on different Discs which the Windows 7 Installer will ask for during installation. Slips7ream automatically places the optimal amount of WIMs on each Disc depending on file size, and includes a tiny AutoRun program to let you know it's a Data Disc, not the Setup Disc.
