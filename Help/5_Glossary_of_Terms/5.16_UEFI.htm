Section 5.16

UEFI

The Unified Extensible Firmware Interface is a replacement to the ancient concept of a BIOS or CMOS on a PC Motherboard. UEFI only works with 64-Bit Windows, and can not read the UDF File System or files larger than 4095 MB during setup. Windows 7 also does not have the drivers to support a video card under UEFI, so a Legacy Graphics or Compatibility Module option is still required. Simply enable the required feature and boot to the installation media through UEFI specifically.

Slips7ream can enable UEFI support in an ISO by copying some EFI-support files to the right folders and ensuring the File System can be read by the UEFI. If you can find and integrate the right drivers, installation may be successful, but Safe Mode and other things that rely on vga.sys will fail.
