Section 5.18

Windows Update

A Windows Update is a patch for the Windows Operating System which includes a fix or improvement to core files. Technically these include Security Updates, Internet Explorer Updates, and Language Packages. Updates are the reason why Slipstreaming exists: so they don't have to be installed later by hand.

Most Updates contain a Knowledge Base article or other Support Link to view details about what the update does. Slips7ream uses the Integrated Windows Update list, if populated, when Adding Updates to let you know of any duplicates, missing requirements, updates, or already integrated newer versions.
