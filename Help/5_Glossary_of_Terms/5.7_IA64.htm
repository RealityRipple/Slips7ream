Section 5.7

IA64 / Itanium

Intel's IA64 processor is a server-oriented competitor to AMD's more successful x64 standard. Windows 7 does not support the IA64 architecture - since Itanium processors were geared toward servers, Windows Server 2008 was the Operating System to support IA64.

Some Drivers may contain IA64 support, but the Architecture dropdown will be marked as "Not Used". The Driver INF may be included if it contains other architectures, but it will not be used in Driver or Integrated Driver comparisons.
