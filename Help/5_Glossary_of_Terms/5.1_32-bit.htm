Section 5.1

32-bit / x86

The x86 architecture was made for IBM's 286 processor and is well known for the 32-bit limitation of what's essentially 4 GB. Many File Systems still suffer from this limitation, including ISO/9960.

Windows 7's 32-bit flavor has been fully supported by Microsoft, and in fact the "Starter Edition" is only available in 32-bit, likely due to the fact that 64-bit Operating Systems contain all the files of the 32-bit ones for full compatibility with 32-bit programs.

SLIPS7REAM uses the Update's Properties to determine its architecture and match it against every Image Package in the selected Image(s), allowing for quick updating of All-In-One packages or other images with mixed architectures.

The x86 Update Whitelist allows 32-bit updates to be installed on 64-bit Operating Systems when necessary, such as the Spelling and Hyphenation Packs for Internet Explorer. However, this is not a reversible process. 64-bit Updates, Drivers, Applications, and Language Packs will never work on 32-bit Operating Systems, and similarly, Windows 7 Service Pack 1 requires a 64-bit Operating System when integrating into a 64-bit Image Package.
