Section 5.4

DISM

The Deployment Image Service and Management tool is a command-line program built into Windows. It can modify Image Packages and the active Operating System by installing and removing Windows Updates, Drivers, Language Packs, and Windows Features. It can also provide detailed information about the Image Packages or files.

Slips7ream is basically a Graphical User Interface for DISM, designed for use with multiple Image Packages in a single WIM. If you want to see the inner commands DISM receives from Slips7ream, you can use the Output Console.

Slips7ream uses the Dism.exe file inside your Operating System's "System32" folder, unless the version is below 6.1.7600.16385.
