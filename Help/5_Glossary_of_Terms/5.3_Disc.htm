Section 5.3

Disc / ISO

An ISO or Disc Image is an exact copy of the data on a CD, DVD, or similar storage media. They're a common way to backup actual Discs onto hard drives, and many Operating System makers such as Microsoft and the Linux community use ISO files to distribute their products.

Slips7ream can read ISO files using 7-zip, and create them using OSCDIMG, allowing you to manipulate ISOs or WIMs interchangeably. It can also Split up the Windows 7 Installer into multiple ISOs. Slips7ream does not, however, generate Windows 7 installer data automatically, so when saving to an ISO, it is important to select an ISO that already exists for Slips7ream to place the WIM into. It's also important to use a 32-Bit ISO when making an All-In-One or mixed architecture installer, because 32-Bit processors can not read 64-Bit files.
