Section 5.5

Driver

A Driver INF File is basically an identification for a piece of hardware, often with software included to make use hardware functions. 64-Bit Operating Systems require Drivers to be digitally signed by default; however SLIPS7REAM can still integrate unsigned Drivers for user convenience.

SLIPS7REAM allows integration of Drivers that are almost identical, including newer and older versions of Drivers, because Windows stores backups for Rollback and other compatibility resolutions. Some Drivers also may include important Hardware IDs that newer or older versions do not. SLIPS7REAM will, however, warn you if you try to integrate an identical Driver.
