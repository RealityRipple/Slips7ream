Section 5.17

Windows Feature

Windows Features are programs, components, and services that are optionally included with Windows. Some popular Features are Internet Explorer, Outlook Express, Solitaire, and IIS.

Slips7ream allows you to turn Features on and off through the Package Properties window. Turning off features by default can save time and hard drive space during Installation, or you can customize an install for a specific home or workplace environment setup by adding helpful features to start with.
