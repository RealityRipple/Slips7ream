Section 5.12

Package

In SLIPS7REAM, Package usually refers to an Image Package containing a copy of the Windows 7 file structure.

In some instances, particularly in the context of the DISM command-line tool, a Package is a Windows Update or Language Pack.

SLIPS7REAM tries to use "Image Package" all the time, but some disambiguation may occur. Context will hopefully clarify any possible confusion.
