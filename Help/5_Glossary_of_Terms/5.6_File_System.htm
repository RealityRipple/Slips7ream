Section 5.6

File System

A File System is a backend structure for the organization of data on a storage medium. Slips7ream only worries about Disc File Systems such as ISO 9960, Joliet, and UDF.

The biggest difference between File Systems is the maximum file size. Slips7ream will warn you about any files that are larger than the 4 GB limit if you attempt to make a Disc with a File System that suffers from that limit. You can also Split a WIM into parts to prevent large files. This is especially important for UEFI compatibility.
