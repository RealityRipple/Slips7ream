Chapter 4

Included External Tools

SLIPS7REAM, at its core, is a graphical interface for a series of text-based programs which allow manipulation of WIM files. While it performs many tasks internally, the bulk of the important work is done by these command-line tools.

The tools are stored in a "tools" folder in the SLIPS7REAM installation directory, and include both x86 and x64 variants. They are vital for the functionality of SLIPS7REAM.
