Section 4.3

OSCDIMG

oscdimg.exe, or the Microsoft CD/DVD Premastering Utility, is a command-line tool for creating ISO files. SLIPS7REAM uses it to build the final ISO if the Save to ISO checkbox is checked.

Information about OSCDIMG is available from TechNet .

OSCDIMG is part of the Windows Automated Installation Kit, available from Microsoft's Download Center free of charge.

SLIPS7REAM 3.0 uses oscdimg.exe v2.55.
