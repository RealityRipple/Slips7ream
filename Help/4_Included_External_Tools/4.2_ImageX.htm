Section 4.2

ImageX

imagex.exe, or the Windows Imaging Utility, is a command-line tool for working with WIM files themselves. SLIPS7REAM uses it for compressing and splitting WIM files and for updating the language data in a WIM.

Information about ImageX is available from TechNet .

ImageX is part of the Windows Automated Installation Kit, available from Microsoft's Download Center free of charge.

SLIPS7REAM 3.1 uses imagex.exe v6.1.7600.16385.
