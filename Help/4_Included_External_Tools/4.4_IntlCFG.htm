Section 4.4

IntlCFG

intlcfg.exe, or the International Settings Configuration Utility, is a command-line tool for changing the language and locale, fonts, and input settings for a Windows Image. SLIPS7REAM uses it for updating the language data in a WIM.

Information about IntlCFG is available from TechNet .

IntlCFG is part of the Windows Automated Installation Kit, available from Microsoft's Download Center free of charge.

SLIPS7REAM 3.0 uses intlcfg.exe v6.1.7600.16385.
