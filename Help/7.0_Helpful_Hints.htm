Chapter 7

Helpful Hints

  • You usually only need the latest Internet Explorer update as they tend to be rollups. Sometimes a second update is necessary, but this is rare.
  • Don't bother integrating KB2952664, KB3021917, KB3068708, or KB3080149. These updates are simply telemetry and Windows 10 upgrade related.
  • Getting a "System Update Readiness Tool" update listed in your fresh install's Update list? Keep a copy of KB947821 handy and install it after Windows is installed (KB947821 can not be integrated).
  • When in doubt, use the Microsoft Update Catalog to get the latest Updates. The Download Center often has outdated or non-Slipstreamable Updates (such as KB2647753-v2, while v4 can be Slipstreamed).
  • UEFI Booting is useless unless you integrate video card drivers into your installer. The standard VGA.SYS driver that comes with Windows 7 is not UEFI compatible.
  • The latest Windows Update Agent (KB949104 ) can be integrated into your installer so you can quickly and safely check for Updates immediately after installation.
