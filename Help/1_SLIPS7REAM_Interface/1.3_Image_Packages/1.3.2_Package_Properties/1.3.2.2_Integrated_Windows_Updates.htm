Section 1.3.2.2

Integrated Windows Updates

The Integrated Windows Updates list can be found to the right of the Package Properties list, below Windows Features. This list includes all the Updates and Language Packs which have already been integrated into the Image Package.

You can populate the list of Updates by clicking on the "Load Integrated Windows List" button, or by using the Parse Packages feature.

Once the list has been loaded, you can uncheck the checkbox next to any Update which you would like to remove. It's important to note that Updates with the State "Uninstall Pending" or "Superseded" can not be removed this way, but will be removed during installation.

Please note that you must run the Slipstream Process before any settings will be changed.
