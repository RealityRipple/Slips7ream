Section 1.3.2

Package Properties

Each Image Package has a range of detailed properties describing the Package. SLIPS7REAM allows you to view all of these properties through an advanced Image Package Properties Window. Below is a description of each entry.

  • Index: This is the index of the Image Package in its parent WIM file. Indexes start from 1. This is the order in which the Installer will list your Operating System choices.
  • Name: The name by which the Image Package is identified when working with WIM files, and what will be shown in the list of Operating Systems during Installation. The Name can be edited to be anything you want, but it must be unique in the Image, so please make sure that no two Packages use the same Name.
  • Description: A simple description of the Operating System, used by the Installer. You can set this to whatever you want to be shown below the list of Operating Systems when this Package's Name is selected.
  • Size: The total size of the Image Package once extracted, not the actual file size of the compressed Image Package.
  • Architecture: This identifies the architecture of the Windows 7 Image Package as either x86 or x64.
  • HAL: The Hardware Application Layer used by the OS, which, as far as I know, is always "acpiapic".
  • Version: The Operating System Version for Windows 7 is always 6.1. Windows 7 RTM is 6.1.7600, Service Pack 1 is 6.1.7601. No other versions currently exist.
  • Service Pack Level: This value is 0 for Windows 7 RTM and 1 for Service Pack 1.
  • Service Pack Build: The Build number is usually seen at the end of the Version (above) in a full OS Version.
  • Edition: The Edition of Windows 7 contained in this Image Package.
  • Installation: The target Installation type for this Operating System. Windows 7 is always "Client".
  • Product Type: The product type of Windows 7 is always "WinNT".
  • Product Suite: The product suite of Windows 7 is always "Terminal Server".
  • System Root: This is the folder where all the Windows files are stored. Windows 7 uses "WINDOWS", though this may be modifiable.
  • File Count: The total number of files contained in this Image Package.
  • Directory Count: The total number of folders contained in this Image Package.
  • Created: The date this Image Package was created. This is the date shown by the Installer.
  • Modified: The date this Image Package was last modified. This date is not shown in the Installer.
  • Languages: This is a list of all the languages which are integrated into this Image Package.
