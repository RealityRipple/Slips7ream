Section 1.3.2.1

Windows Features

The Windows Features list can be found to the right of the Package Properties list. This list is almost identical to the "Turn Windows features on or off" interface in Windows, with a few minor changes.

You can populate the list of features by clicking on the "Load Windows Features List" button, or by using the Parse Packages feature.

Once the list has been loaded, you can select the features you want enabled or disabled by default in this Operating System. The list is designed to enable any required features when you enable a feature that has requirements, and to disable features if a requirement is disabled. You may also receive notices about features being required for parts of the Operating System to run.

Please note that you must run the Slipstream Process before any settings will be changed.
