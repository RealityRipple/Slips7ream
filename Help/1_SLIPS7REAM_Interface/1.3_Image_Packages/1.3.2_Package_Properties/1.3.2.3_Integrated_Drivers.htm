Section 1.3.2.3

Integrated Drivers

The Integrated Drivers list can be found to the right of the Package Properties list, below Integrated Windows Updates. This list includes all the Drivers which have already been integrated into the Image Package.

You can populate the list of drivers by clicking on the "Load Integrated Drivers List" button, or by using the Parse Packages feature.

Once the Driver Class list has been loaded, you will be able to select one of the Classes to get a list of Providers and INF Files. Choosing a different Class or Provider will list the INF Files that match. You can uncheck an INF file to remove it from the Image Package. Double-Clicking an INF File will display the Driver's Properties.

If you come across a Provider that is missing its icon (showing instead ), please contact me and I will include a company logo for the Provider in the next version of SLIPS7REAM.

Please note that you must run the Slipstream Process before any settings will be changed.
