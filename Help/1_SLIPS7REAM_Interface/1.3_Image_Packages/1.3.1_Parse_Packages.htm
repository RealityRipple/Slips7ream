Section 1.3.1

Parse Packages

Next to the Image Packages List is a series of four buttons with icons. The first three of these are options to select which information will be parsed from the Image Packages in the Image Packages list. The final button begins the Parse procedure.

  •  This option enables loading the Windows Features from the packages.
  •  This option enables loading the Integrated Windows Updates from the packages, which will be used to prevent duplicate updates from being added to the Updates List.
  •  This option enables loading the Integrated drivers from the packages.

When you're ready to begin the Parse process, click on the " button". SLIPS7REAM will automatically go through every checked Image Package and read the requested data from them. When the process is complete, the icon will change to a .

Please note that this feature is entirely optional. The total load time may be quite a while depending on how many Windows Updates and Drivers have been integrated, so be careful when enabling all of these options at once.
