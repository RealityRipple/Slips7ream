Section 1.3

Image Packages

The Image Packages list is a list of all the Windows Installation Image Packages that are contained within the INSTALL.WIM and Merge WIM files - if the Merge WIM checkbox is checked, the list is separated into two groups: "Source Image" and "Merge Image". Details about a package can be found by double-clicking on it or right-clicking it and choosing the Properties menu item.

If an Image Package checkbox is unchecked, that package will not be included in the Slipstream Process. Packages can be renamed, but the name will only show up in SLIPS7REAM or other Disc Image Servicing tools, not in the Windows Installer.

You can change the display order of the Image Packages by clicking on the Index, Name, or Size column headers. Each column will sort in either Descending or Ascending order with a second click using the following methods:

  • Index: The Image Packages will be sorted by their Index numbers, grouped by the WIM they came from. This is the default sort order, and the only sort order that separates the INSTALL.WIM from the Merge WIM packages.
    Note that the Ascending order will put the Merge WIM group first.
  • Name: This sort order is a little more complex than it sounds. Image Packages will be organized by their Architectures, Service Pack numbers, OS Type (Starter through Enterprise), and finally last modification date.
  • Size: The simplest of the sort orders, this one organizes the Image Packages by their compressed size.
