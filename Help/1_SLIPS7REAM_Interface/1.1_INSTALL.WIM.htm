Section 1.1

INSTALL.WIM

The INSTALL.WIM field accepts a Windows Image or ISO (if the ISO contains a Windows Image in the "\sources\" folder).

WIM File Type

If this is a WIM, then all changes are applied directly to the WIM file.
If you want to keep an unchanged original of the WIM you're working with, make a copy of the original source WIM and select the copy.

ISO File Type

If this is an ISO, then the WIM is extracted to your Temp Directory and modified there.
It is recommended that you use the Save to ISO to select this or another ISO to save the finalized WIM to, otherwise the WIM will stay in the Temp Directory and will be deleted when the program is closed or another operation begins.

Note: If you select a WIM-type file, enabling Save to ISO does not prevent changes to the file selected in this field. If you use an ISO-type file for this field, then the changes will only be applied to the ISO file in the Save to ISO field, and this ISO will remain unmodified.
