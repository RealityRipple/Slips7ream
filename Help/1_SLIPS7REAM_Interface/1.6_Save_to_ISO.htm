Section 1.6

Save to ISO

The Save to ISO field accepts a Windows Installer ISO which will store the INSTALL.WIM once all operations are complete.

This ISO file must already exist, but as is the case when working with a WIM, a copy should be used if you wish to keep the unchanged original ISO.

The contents of the ISO (minus any WIM or SWM files) will be extracted and modified as per the ISO Features and a new ISO will be made with the finalized WIM.

If the Read ISO checkbox is checked, the ISO Label is automatically read from the ISO you select. You can still enter another ISO name, but entering a new ISO into the Save to ISO field will change the name again.

Note: If you select a WIM-type file in the INSTALL.WIM field, then enabling this feature does not prevent changes to the file selected in the INSTALL.WIM field. If you use an ISO-type file, then the changes will only be applied to the ISO file in this field, and the INSTALL.WIM ISO will remain unmodified.
