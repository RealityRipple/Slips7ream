Section 1.5.1

Add Updates

You can add Updates, Language Packs, and Drivers to the Updates list by clicking the "Add Updates..." button or by dragging files or folders into the list. SLIPS7REAM supports Windows Updates in MSU or CAB format, Language Packs in MLC, EXE, MSI, or LP.CAB format, and Drivers in INF format.

SLIPS7REAM will also compare files in the Updates list and the Integrated Packages lists. If any update or language pack is already integrated, SLIPS7REAM will prompt you, either with a warning about its inclusion or by letting you select the version you wish to use if they are different versions. Drivers are compared by Class GUID, Date, Version, Architectures, and Hardware IDs (ignoring IA64 architectures and Hardware IDs) because drivers don't interfere with each other.

There are certain caveats and issues when adding Updates:

  • Internet Explorer integration has prerequisites which are not included in the Setup EXE.
    Internet Explorer 10 requires:
    1. KB2729094
    2. KB2731771
    3. KB2533623
    4. KB2670838
    5. KB2786081
    6. KB2639308
    Internet Explorer 11 requires:
    1. KB2729094
    2. KB2731771
    3. KB2533623
    4. KB2670838
    5. KB2786081
    6. KB2639308
    7. KB2834140
    8. KB2888049
    9. KB2882822
  • The .NET Framework 4.0 and above can not be integrated.
  • The Malicious Software Removal Tool can not be integrated.
  • Windows Defender updates can not be integrated.
  • The Update Readiness Tool (KB947821) can not be integrated.
  • KB2533552 (now defunct) can not be integrated.
  • KB3035583 should not be integrated (it just notifies you about Windows 10 anyway).
  • KB2647753 v2 does not integrate correctly. You can get v4 (or above) from the Microsoft Update Catalog .
  • KB2830477 requires KB2574819 and 2857650.
  • The Language Packs for ca-ES, cy-GB, hi-IN, is-IS, and sr-Cyrl-CS have been superseded by Service Pack 1 and do not integrate into SP1 Images correctly.

There are also requirements for most Language Packs:

  • The Language Packs for af-ZA, sq-AL, am-ET, hy-AM, as-IN, bn-BD, bn-IN, prs-AF, fil-PH, ka-GE, gu-IN, ha-Latn-NG, hi-IN, is-IS, ig-NG, id-ID, iu-Latn-CA, ga-IE, kn-IN, km-KH, sw-KE, kok-IN, mk-MK, ms-MY, ms-BN, ml-IN, mt-MT, mi-NZ, mr-IN, ne-NP, or-IN, fa-IR, pa-IN, nso-ZA, tn-ZA, si-LK, ta-IN, te-IN, ur-PK, vi-VN, cy-GB, xh-ZA, yo-NG, and zu-ZA require the English Language Pack.
  • The Language Packs for gl-ES and quz-PE require the Spanish Language Pack.
  • The Language Pack for nn-NO requires the Norwegian (Bokmål) Language Pack.
  • The Language Packs for ky-KG and tt-RU require the Russian Language Pack.
  • The Language Packs for eu-ES and ca-ES require either the French or Spanish Language Pack.
  • The Language Packs for bs-Cyrl-BA and bs-Latn-BA require either the English, Croatian, or Serbian (Latin) Language Pack.
  • The Language Pack for sr-Cyrl-CS requires either the English or Serbian (Latin) Language Pack.
  • The Language Pack for lb-LU requires either the English or French Language Pack.
  • The Language Packs for az-Latn-SZ, kk-KZ, mn-MN, tk-TM, and uz-latn-UZ require either the English or Russian Language Pack.
