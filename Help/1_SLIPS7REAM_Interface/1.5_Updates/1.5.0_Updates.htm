Section 1.5

Updates

The Updates list includes all Windows Updates, Language Packs, and Drivers that are going to be integrated into each Image Package. The order of the items in the list can be changed by dragging items around or by right-clicking items and using the context menu options.

If there is an issue with an Update, the color of the list item's text will be different: Details about the issue can be found in the Tooltip of the item if you hover the mouse over it for a moment. Red text means that there is an integration issue, and Orange text means that a requirement may not be included.

Drivers have an extra feature within the Updates list. If you use the Save to ISO feature, you'll see two context menu items for Drivers that aren't normally there: "Include Drivers in PE" and "Include Drivers in Setup". As suggested, this allows you to integrate proprietary drivers into your BOOT.WIM file. Please note that "PE" is the Windows Setup Pre-Environment, while "Setup" is used for repairs. Confusing, I know, but that's the way Microsoft named them. The color of the driver's text will change depending on what you select: Blue for PE, Green for Setup, and Purple for both.
