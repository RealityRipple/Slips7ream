Section 1.9

On Completion

When the Slipstream Process is completed, SLIPS7REAM can automatically perform one of the following actions:

  • Do Nothing: SLIPS7REAM will not do anything when complete. It will just show its normal interface again.
  • Play Alert Noise: The Alert selected in the Configuration Window will be played. If no Alert is selected, the Windows "tada.wav" or System Default Alert sound is used instead.
  • Close Program: SLIPS7REAM will close itself.
  • Shut Down: Windows will be shut down and the computer turned off.
  • Restart: Windows will be restarted.
  • Sleep: Windows will be sent into sleep mode (also called Standby or Suspend).
