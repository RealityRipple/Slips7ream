Section 1.10.8

File could not be played

If you receive an error dialog that says "The Alert sound file failed to play!", then the selected file may not be supported or may be corrupt. Clicking on the Details button will provide you with specific information on the nature of the failure.

If you receive an error dialog that says "The Alert sound file does not exist!", then the file you have selected may be missing or inaccessible by SLIPS7REAM. Please check the file path and permissions to ensure the file can be read.
