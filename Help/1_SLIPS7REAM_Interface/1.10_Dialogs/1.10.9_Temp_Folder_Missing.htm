Section 1.10.9

Unable to find parent directory

If you choose a Temp directory path that doesn't exist, SLIPS7REAM will prompt you to select one. The Temp directory is where SLIPS7REAM does all its work, so please select a location that can handle at least a dozen gigabytes of temporary data.

Note that the "Slips7ream" folder in the Temp directory will be created and deleted by the program repeatedly - it's the parent directory that has to exist. By default, SLIPS7REAM uses your computer's TEMP directory: "C:\Users\[USERNAME]\AppData\Local\Temp\".
