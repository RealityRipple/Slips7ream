Section 1.10.11

Error loading packages

You may receive an error dialog while loading files that are already integrated into the Image Package. This error occurs if there is a failure for any reason. Pressing the "Cancel" button on the Main window will also cause this error to show up: if you cancelled the parse process, you can ignore this error.

If you receive an error dialog while attempting to load the Windows Features List, you will need to check the Output Console to see what went wrong, but the gist is that something prevented SLIPS7REAM from reading the Features list from your Image Package.

If you get an error loading Integrated Windows Updates, something stopped SLIPS7REAM from reading the Integrated Updates from your Image Package. You can check the Output Console to find out what the problem is.

When loading Integrated Drivers, you may receive an error dialog if a problem occurred. The Output Console will provide you with details regarding the issue.
