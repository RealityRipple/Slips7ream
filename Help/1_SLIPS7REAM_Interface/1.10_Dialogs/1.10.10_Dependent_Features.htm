Section 1.10.10

Dependent Windows features will be turned off

When disabling Windows Features in the Image Packages window, you may receive a dialog that notifies you that other Windows features and programs on your computer might also be affected if you disable the feature you're trying to disable. If you don't want to continue, you can hit "No" and the feature will remain enabled. In many cases, a hyperlink in the dialog will provide a page with more information from Microsoft.

If you enable a feature and receive an error dialog that says "A required feature is missing from the feature list", you may have a modified or damaged Windows Image Package, or the Feature list may not have been completely or correctly loaded. In either case, a feature that is required for the selected feature could not be found in the list, so the selected feature will not be enabled.
