Section 1.10.6

Unable to enable UEFI Boot

If you have enabled UEFI Boot, a few EFI-related files need to be copied to a new location in the ISO. If those files are missing from the original ISO, you may receive this error. If this error does occur, SLIPS7REAM will continue with the integration process without enabling UEFI.

If "bootmgfw.efi" can not be found, SLIPS7REAM will prompt you with an EFI File Not Found error dialog. The error will not interfere with the Slipstream process, but it will prevent the UEFI Boot feature from being enabled.

If the "\EFI\MICROSOFT\BOOT\" folder can not be found, SLIPS7REAM will prompt you with an EFI Folder Not Found error dialog. The error will not interfere with the Slipstream process, but it will prevent the UEFI Boot feature from being enabled.
