Section 1.10.1

Unable to add files to the Update List

You may receive this error while adding files to the Update List. The "View Details" button on the dialog will provide specific details.

The reasons for an update not being included are:

  • Description/Properties File Not Found: the file that describes the update (usually an "update.mum" inside a compressed file) was not found in the expected location. This is often a sign of a corrupt file or a file that isn't supported. Other Description file errors may occur with the same general meaning and further details as to the nature of the problem.
  • Errors during file extraction such as CRC Errors, Data Errors, Unsupported Method, File Not Found and other Extraction Error messages.
  • Unknown file type: Slips7ream only allows the following file types: CAB, MLC, MSI, MSU, EXE, and INF. Other file types are not supported.
  • Update not found: the file doesn't exist. It may have been deleted or moved, or the drive or path may not be accessible.
