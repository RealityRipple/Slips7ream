Chapter 1

SLIPS7REAM Interface

The Interface of SLIPS7REAM may look daunting to begin with, but its features are clean, organized, and to the point.

The interface is split up into three main parts: the Operating Systems, the Updates, and the ISO options.
The top of the interface is a large blue bar which displays any operation in progress.
Below that are the INSTALL.WIM and Merge WIM fields, the Image Packages list, and the Service Pack entries. These items all reside above a resizable splitter in the middle of the interface.
The Updates field below this splitter let you add, remove, organize, and view details about Windows Updates, Drivers, and Language Packs that will be added to all supported Image Packages.
The Save to ISO field and the following ISO Features let you save your INSTALL.WIM to an ISO Image which can be burned directly to a Disc.
Finally, a few remaining features such as the Process Priority and On Completion fields round out the interface, with just enough space for the Debug arrow, Activity label, and Configuration button.
