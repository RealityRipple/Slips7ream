Section 1.7.4

File System

This field chooses the File System which will be used for the Disc. If you have any files that are larger than 4 GB, then you should use a UDF file system.

  • ISO 9960: Standard format. Limited file name conventions.
    Details: <https://en.wikipedia.org/wiki/ISO_9660 >.
  • Joliet: Joliet extension of ISO 9960, endorsed by Microsoft. Enhanced file name capabilities.
    Details: <https://en.wikipedia.org/wiki/Joliet_(file_system) >.
  • UDF: Universal Disk Format, or ISO/IEC 13346. Superior standard, but less compatible. Large file sizes.
    Details: <https://en.wikipedia.org/wiki/Universal_Disk_Format >.
  • UDF 1.02: Legacy UDF understood by most drives. Compatible with old hardware.

Items marked with "/ISO 9960" employ a backup format using the ISO 9960 standard when the selected format can't be read.
