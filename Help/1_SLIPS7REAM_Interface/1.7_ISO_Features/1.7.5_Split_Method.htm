Section 1.7.5

Split Method

The Split Method chooses how files will be split to fit your installer on limited disc space. This is the only ISO Feature which applies to WIM files as well, and therefore does not require the Save to ISO checkbox to be checked.

There are three options to choose from:

  • Single File: The WIM will not be split and all data will be placed into a single ISO.
  • Split WIM: The WIM will be split into smaller SWM files, each of which will be the size you define, at most (see Size Limit below). Each WIM will be stored on one ISO file. This can be particularly useful if you need an ISO 9960 File System but have a WIM larger than 4 GB.
  • Split ISO: The WIM will be split into smaller SWM files which will each be contained in smaller ISO files. The ISO files will be the size you define, at most (see Size Limit below), while the WIM files will be about 350 MB smaller allowing for installer data. In some cases, a disc may contain more than one SWM. This option is only available if the Save to ISO checkbox is checked.

Size Limit: This field determines the maximum size of the SWM or ISO file. A few default values are included in the list to help you choose a limit automatically, but any value can be entered manually. All sizes are converted to MB after being entered. The size can not be less than 351 MB if the Split Method (see Above) is set to "Split ISO".
