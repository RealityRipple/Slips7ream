Section 1.7.1

ISO Label

This label is used when creating the ISO and can be anything you want. It will be used as the Disc Label if you Mount or Burn the image.

Any ISO file can be dragged into this field to automatically load the label of that ISO. You can also right-click this field to get a list of Windows 7 standardized labels you can use, or select Auto-Detect to let SLIPS7REAM determine what label best suits your image.

If the Read ISO checkbox is checked, the ISO Label is automatically read from the ISO you select. You can still enter another ISO name, but entering a new ISO into the Save to ISO field will change the name again.

Limit: 32 characters.
