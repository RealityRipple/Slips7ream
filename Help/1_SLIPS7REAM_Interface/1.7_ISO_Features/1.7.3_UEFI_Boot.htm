Section 1.7.3

UEFI Boot

The UEFI Boot checkbox enables installation on UEFI motherboards without requiring a BIOS Compatibility Module.

Please note that Windows 7 does not contain the drivers required for native UEFI support. If you can enable legacy graphics in your UEFI without enabling full compatibility, or if you integrate your Video Card Drivers into Windows 7, you can install Windows 7 through UEFI successfully, but the default "vga.sys" driver is not UEFI compatible, so no matter what, Safe Mode and other instances of default display drivers will fail to display anything.
