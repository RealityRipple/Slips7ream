Section 1.7.3

UEFI Boot

The UEFI Boot checkbox enables installation on UEFI motherboards without requiring a Legacy BIOS Compatibility Support Module.

Please note that Windows 7 does not contain the drivers required for native UEFI support. If you integrate your Video Card Drivers into Windows 7, you can install Windows 7 through UEFI successfully. However, the default "vga.sys" driver is not UEFI compatible, so even with a compatible driver, Safe Mode and other instances where the default display drivers are used will fail to display anything. The better option, if you can, is to enable Legacy Graphics/Video Support in your UEFI without enabling full Legacy BIOS Compatibility.
